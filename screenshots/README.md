# Screenshots

GitHub cannot render PBIX/PBIT report pages. Add PNG exports created directly in Power BI Desktop to this folder.

Recommended evidence set:

1. One full-page screenshot for each report page.
2. One close-up showing slicer interaction or drill-down.
3. One model or measure screenshot when appropriate.

Do not create mock screenshots or use images that show values not present in the report. Remove private paths, account details and confidential institutional information before publishing.
